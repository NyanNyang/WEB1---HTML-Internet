JavaScript (/ˈdʒɑːvəskrɪpt/[6]), often abbreviated as JS, is a high-level, dynamic, weekly typed, prototype-based, multi-paradigm, and interpreted programming language. Alongsite HTML and CSS, JavaScript is one of the three core technologies of World Wide Web content production. It is used to make webpages interactive and provide online programs, including video games. The mejority of websites employ it, and all modern web browers support it without the need for plug-ins by means of a built-in JavaScript engine.
